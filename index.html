POPout is a software package that allows users with PRS data to identify and quantify the enrichment of large effects in one or both of the tails of the trait distribution.

Overview

  • POPout code is available in R and in Python3. Users can use either version.
  • For more information on installing dependencies, please refer to Requirements.
  • To run POPOut using toy data see our Quick Start Tutorial..
  • Following the Quick Start, the full guide provides more information on the theory and the options regarding POPout.
